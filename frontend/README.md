# Frontend part of tech-prog-app

## Technologies:

1.  React
2.  Redux Toolkit
3.  React-router-dom v6

## Запуск приложения в режиме разработки:

0. Перейти в директорию "frontend" (из корня приложения)
   ```bash
   cd frontend
   ```
1. Установка зависимостей
   ```bash
   npm install
   ```
2. Запуск
   ```bash
   npm run dev
   ```
